It's a config file that 
1. compilerOptions: 
	1. outDir - where to put compiled files.
	2. declaration:  false | true - whether to create declaration file.
	3. target - what version of JS to compile the typescript: ES2015, ES2017, ES2022.
	4. moduleResolution
2. include - root folder, what to compile
